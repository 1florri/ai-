## 3. 核心技术实现
本项目采用原生 HTML + CSS + JavaScript，不依赖 Node、Vue 或 React，保证在 Live Server 环境下可直接运行。整体结构分为顶部操作栏、中部聊天区、底部输入区三块，并通过 CSS 变量管理浅色/深色主题。主题按钮点击后会切换 `body` 的类名，并将结果写入 `localStorage`，刷新页面后读取恢复。

对话功能方面，前端调用阿里云百炼兼容接口 `compatible-mode/v1/chat/completions`，请求参数开启 `stream: true`。通过 `fetch + ReadableStream` 读取服务端逐段返回的数据，解析每一行 `data:` 消息，提取 `delta.content` 并实时拼接到 AI 回复中，实现类似打字机的流式展示。停止按钮使用 `AbortController` 主动取消当前请求，防止长回答继续输出。

Markdown 渲染采用前端自定义解析流程：先提取代码块，再对标题、列表、引用、行内代码、表格进行转换，最后回填代码块。代码块区域附带复制按钮，点击后写入剪贴板并显示反馈提示。图片上传使用浏览器本地 `URL.createObjectURL` 生成预览图，便于展示交互流程。清空对话会回收临时 URL、清空消息数组并恢复欢迎区，保证状态干净可复用。
